O que é o HTML

O html é uma linguagem de marcação utilizada para o desenvolvimento de páginas web.

Dentre as tags de marcação nós temos os exemplos abaixo:

  • p: Principal tag de texto, compõe um parágrafo;
  • span: Legendas para formulários, imagens etc;
  • pre: Texto pré formatado. (Inserir códigos);
  • b: Negrito;
  • i: Negrito;
  • br: pula linha horizontal.

Tipos de site

Com a linguagem de marcação html podemos criar vários tipos de site. Vamos verificar alguns do exemplos disponíveis.

  • Site Institucional: Site institucional é utilizado pela maioria das empresas. Ele é composto por algumas páginas com informações sobre seu negócio; Ver Exemplo

  • Site dinâmico: Este projeto torna um site mais dinâmico. Permite alterações de maneira fácil. O conteúdo pode ser atualizado constantemente; Ver Exemplo

  • Lojas Virtuais (e-commerce): Site de comércio eletrônico, ambiente online onde podem ser realizadas vendas de produtos físicos ou virtuais; Ver Exemplo

  • Blogs:Site dinâmico porém possuí um layout voltado para leitores. Onde temos informações sobre algo de um nicho em comum, como filmes, séries, dicas de culinária, literatura etc...; Ver Exemplo

  • Portais de Notícias:Sites onde conseguimos agrupar conteúdos diversos. Geralmente seu foco é voltado a um público específico, onde as pessoas buscam acessar diariamente para se atentar a notícias ou referência de determinado assunto. Normalmente aplicado a cidades locais/regionais e assim por diante. Ver Exemplo
